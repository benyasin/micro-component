# Micro Components 概览

- 跨栈运行时：支持在 Vue2 / Vue3 / React 中以统一 API 使用
- 动态加载：运行时按需加载组件与多语言资源
- 事件通信：useEvent 提供跨宿主事件总线
- 国际化与主题：可全局开关，组件可单独开启

## 快速开始

1. 安装并启动 playground 进行开发
2. 在宿主应用里引入运行时 `dist/micro-runtime.js`
3. 在 Vue/React 项目中按模板使用对应适配层

> 详细设计与约定请参考项目 README。
