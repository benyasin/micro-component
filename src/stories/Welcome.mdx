# Welcome to Micro Components

欢迎使用 Micro Components 组件库！这是一个跨框架的微组件库，支持在 Vue2、Vue3 和 React 中使用。

## 特性

- **跨栈运行时**：支持在 Vue2 / Vue3 / React 中以统一 API 使用
- **动态加载**：运行时按需加载组件与多语言资源
- **事件通信**：useEvent 提供跨宿主事件总线
- **国际化与主题**：可全局开关，组件可单独开启

## 快速开始

### 安装

```bash
npm install micro-components
```

### 使用示例

#### Vue 3

```vue
<template>
  <Footer 
    :locale="locale" 
    :theme="theme" 
    @language-change="handleLanguageChange"
    @theme-change="handleThemeChange"
  />
</template>

<script setup>
import { Footer } from 'micro-components/vue/Footer'

const locale = ref('en')
const theme = ref('light')

const handleLanguageChange = (language) => {
  locale.value = language.locale
}

const handleThemeChange = (newTheme) => {
  theme.value = newTheme
}
</script>
```

#### React

```jsx
import React, { useState } from 'react'
import { Footer } from 'micro-components/react/Footer'

function App() {
  const [locale, setLocale] = useState('en')
  const [theme, setTheme] = useState('light')

  const handleLanguageChange = (language) => {
    setLocale(language.locale)
  }

  const handleThemeChange = (newTheme) => {
    setTheme(newTheme)
  }

  return (
    <Footer
      locale={locale}
      theme={theme}
      onLanguageChange={handleLanguageChange}
      onThemeChange={handleThemeChange}
    />
  )
}
```
