# Test Projects

这个目录包含用于测试编译后组件产物的真实项目环境。

## 项目结构

```
test-projects/
├── vue3/          # Vue 3 测试项目
├── react/         # React 测试项目
├── vue2/          # Vue 2 测试项目
└── scripts/       # 自动化测试脚本
```

## 使用方式

1. **构建组件**: 在根目录运行 `pnpm build` 构建所有框架的组件产物
2. **安装到测试项目**: 运行 `pnpm test:install` 将构建产物安装到各个测试项目
3. **启动测试项目**: 进入对应目录运行 `pnpm dev` 启动开发服务器
4. **运行 E2E 测试**: 运行 `pnpm test:e2e` 执行端到端测试

## 特点

- **真实环境测试**: 使用编译后的产物而非源码
- **多框架支持**: 覆盖 Vue2、Vue3、React 三个主流框架
- **自动化流程**: 支持一键构建、安装、测试的完整流程
- **E2E 验证**: 通过端到端测试验证组件在真实环境下的功能

## 与 playground 的区别

- playground: 开发时使用源码进行调试和预览
- test-projects: 使用编译后的产物进行真实环境测试
